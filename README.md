# Job Application Form Validation

This project is a Job Application Form created using HTML, CSS, and JavaScript as part of my internship task.

## Features
- User-friendly Job Application Form
- Name Validation
- Email Format Validation
- Phone Number Validation
- Message Field Validation
- Error Messages for Invalid Inputs
- Responsive and Clean UI Design

## Technologies Used
- HTML
- CSS
- JavaScript

## Form Fields
- Full Name
- Email Address
- Phone Number
- Cover Message

## Purpose
The purpose of this project is to create a job application/contact form with proper input validation using JavaScript and an attractive user interface using CSS.
